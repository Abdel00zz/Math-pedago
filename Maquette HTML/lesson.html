Suites Numériques

schedule 45 minutes article 12 sections school 2BSM • Semestre 1

Introduction

Les suites numériques sont des fonctions définies sur l'ensemble des entiers naturels ℕ ou une partie de ℕ. Elles constituent un outil fondamental en mathématiques et apparaissent dans de nombreux domaines comme la physique, l'économie et l'informatique.

Dans ce chapitre, nous allons étudier les différents types de suites, leurs propriétés, et les méthodes pour déterminer leur comportement à l'infini.

Définitions Fondamentales

menu_book Définition : Suite numérique

Une suite numérique est une fonction $u$ définie sur ℕ (ou une partie de ℕ) à valeurs dans ℝ.

$u : \mathbb{N} \to \mathbb{R}$
$n \mapsto u(n) = u_n$

On note généralement $(u_n)$ ou $(u_n)_{n \in \mathbb{N}}$ pour désigner la suite.

Modes de Définition

Une suite peut être définie de plusieurs manières :

  • Par une formule explicite : $u_n = f(n)$ où $f$ est une fonction de $n$
  • Par récurrence : On donne $u_0$ (ou $u_1$) et une relation $u_{n+1} = f(u_n)$
  • Par une relation de récurrence : On exprime $u_n$ en fonction de plusieurs termes précédents

lightbulb Exemples

Formule explicite :

$u_n = 2n + 1$ donne $u_0 = 1, u_1 = 3, u_2 = 5, ...$

Par récurrence :

$\begin{cases} u_0 = 1 \\ u_{n+1} = 2u_n + 3 \end{cases}$ donne $u_0 = 1, u_1 = 5, u_2 = 13, ...$

Propriétés des Suites

Sens de Variation

menu_book Définitions

Une suite $(u_n)$ est :

  • Croissante si $\forall n \in \mathbb{N}, u_{n+1} \geq u_n$
  • Décroissante si $\forall n \in \mathbb{N}, u_{n+1} \leq u_n$
  • Strictement croissante si $\forall n \in \mathbb{N}, u_{n+1} > u_n$
  • Strictement décroissante si $\forall n \in \mathbb{N}, u_{n+1} < u_n$
  • Monotone si elle est croissante ou décroissante

info Méthodes pour étudier le sens de variation

  • Étudier le signe de $u_{n+1} - u_n$
  • Si $u_n > 0$, comparer $\frac{u_{n+1}}{u_n}$ à 1
  • Utiliser les variations de la fonction $f$ si $u_n = f(n)$

Suites Bornées

menu_book Définition : Suite bornée

Une suite $(u_n)$ est :

  • Majorée s'il existe $M \in \mathbb{R}$ tel que $\forall n \in \mathbb{N}, u_n \leq M$
  • Minorée s'il existe $m \in \mathbb{R}$ tel que $\forall n \in \mathbb{N}, u_n \geq m$
  • Bornée si elle est à la fois majorée et minorée

Types de Suites Particulières

Suites Arithmétiques

menu_book Définition

Une suite $(u_n)$ est arithmétique s'il existe un nombre réel $r$ (appelé raison) tel que :

$\forall n \in \mathbb{N}, u_{n+1} = u_n + r$

functions Propriétés

Terme général :

$u_n = u_0 + n \cdot r$

Somme des premiers termes :

$S_n = u_0 + u_1 + ... + u_n = \frac{(n+1)(u_0 + u_n)}{2}$

Suites Géométriques

menu_book Définition

Une suite $(u_n)$ est géométrique s'il existe un nombre réel $q \neq 0$ (appelé raison) tel que :

$\forall n \in \mathbb{N}, u_{n+1} = q \cdot u_n$

functions Propriétés

Terme général :

$u_n = u_0 \cdot q^n$

Somme des premiers termes (si $q \neq 1$) :

$S_n = u_0 + u_1 + ... + u_n = u_0 \cdot \frac{1 - q^{n+1}}{1 - q}$

Limites de Suites

menu_book Définition : Limite finie

Une suite $(u_n)$ a pour limite $\ell \in \mathbb{R}$ si :

$\forall \varepsilon > 0, \exists N \in \mathbb{N}, \forall n \geq N, |u_n - \ell| < \varepsilon$

On note : $\lim_{n \to +\infty} u_n = \ell$ ou $u_n \to \ell$

functions Théorèmes Importants

Théorème des gendarmes :

Si $\forall n \in \mathbb{N}, v_n \leq u_n \leq w_n$ et si $\lim v_n = \lim w_n = \ell$, alors $\lim u_n = \ell$

Suites monotones bornées :

Toute suite croissante et majorée converge.
Toute suite décroissante et minorée converge.

Raisonnement par Récurrence

functions Principe

Pour démontrer qu'une propriété $P(n)$ est vraie pour tout $n \geq n_0$ :

  • Initialisation : Vérifier que $P(n_0)$ est vraie
  • Hérédité : Montrer que si $P(n)$ est vraie, alors $P(n+1)$ est vraie
  • Conclusion : La propriété est vraie pour tout $n \geq n_0$

lightbulb Exemple d'application

Montrons que $\forall n \in \mathbb{N}, 1 + 2 + 3 + ... + n = \frac{n(n+1)}{2}$

Initialisation : Pour $n = 0$, $0 = \frac{0 \cdot 1}{2}$ ✓

Hérédité : Supposons $P(n)$ vraie. Montrons $P(n+1)$ :

$1 + 2 + ... + n + (n+1) = \frac{n(n+1)}{2} + (n+1) = \frac{n(n+1) + 2(n+1)}{2} = \frac{(n+1)(n+2)}{2}$

Conclusion : La propriété est vraie pour tout $n \in \mathbb{N}$ ✓

Résumé du Chapitre

assignment Points Clés à Retenir

  • Une suite est une fonction de ℕ vers ℝ
  • Suite arithmétique : $u_{n+1} = u_n + r$ → $u_n = u_0 + nr$
  • Suite géométrique : $u_{n+1} = qu_n$ → $u_n = u_0 q^n$
  • Une suite monotone bornée converge toujours
  • Le raisonnement par récurrence nécessite initialisation et hérédité
